Welcome to currency tracker. An application that retrieves currency exchange rates from a public API and displays them in real-time. Users can select their preferred currencies and track exchange rate fluctuations.

Features

  • Tracks currency with respect to some base currency
  • Convert any amount from one currency to another
  • Remove currencies from the tracked currencies
  • Display the list of the currency that the user is tracking
  • use the help command to view all the commands
